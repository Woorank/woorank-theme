# Install KSS

```bash
$ npm install -g git+https://github.com/kss-node/kss-node.git
```

# Using Style Guide

```bash
$ docker-compose build
$ docker-compose up
```

```bash
$ docker-compose run --rm styleguide gulp
```

# Using gulp (fast developement)

In another terminal window/tab

```bash
$ npm install
$ gulp
```
